# Goal Tracking App

A goal-tracking web application built with Angular that allows users to create, complete, and remove goals through a responsive user interface.

## Features

* Create goals
* Mark goals as completed
* Delete goals
* Responsive user interface

## Technologies Used

* Angular
* TypeScript
* HTML
* CSS

## Running the Application

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Open:

```text
http://localhost:4200
```

## Purpose

This project was developed to strengthen frontend development skills using Angular, TypeScript, and component-based application design.
